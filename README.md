## Executer le projet
    
Cliquez simplement sur index.html et l'ouvrir à l'aide de votre navigateur préféré !

## Edition des fichiers du projet

Pour éditer les fichiers .html / .css il faut les ouvrir avec un éditeur de text.

Soit en faisant clique droit -> Modifier dans Bloc-notes => méthode simple
Soit depuis un éditeur de code adapté comme "VSCode" => méthode avancée
